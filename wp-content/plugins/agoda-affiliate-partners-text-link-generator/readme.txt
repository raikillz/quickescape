This tool was built so that our affiliate partners can easily generate text links in Wordpress.

== Description ==

This WordPress Plugin facilitates the creation of deep links into any page, post or entry, giving Agoda partners an easy way to earn commissions for any booking made through these pages.
All you need to do is select a word or phrase, click the text link icon, and search the city or property you have in mind. The link will be inserted automatically.
Just be sure to log in to Partner Center and get your CID so you can get credit for bookings and earn your well-deserved commission!
If you are not yet an Agoda Affiliate Partner, register now at partners.agoda.com and get your CID immediately.

This plugin use external agoda service to get the city, area, point of interested and property related to keyword.

= Features =

* Select any piece of text and insert a link.
* Easy to use.
* Guaranteed tracking.

The deeper down the booking funnel you link, the higher the conversion rate will be. Our tool lets you search and deep link directly to
1. City,
2. Area,
3. Point of Interest,
4. Specific Hotel/Property

All features are available without leaving your Wordpress Post Editor.

== Installation ==

= First installation: local download =

1. Download the zip file from this page
2. Login to your WordPress dashboard
3. Go to Plugins >> Add New, under Install Plugins title click on Upload and use the browse button to search the .zip file you have downloaded at point 1.
5. The text link creator plugin will happen automatically in post editor

= First installation: download from Wordpress repository directly =

1. Go to Plugins >> Add New
2. Type Agoda Affiliate Partners Text Link Generator into the Search Plugins field
3. Download Agoda Affiliate Partners Text Link Generator plugin

== Frequently Asked Questions ==

= How do I ensure I will earn commissions? =
When using the tool, insure you have entered your CID correctly. The link itself goes to our tracking service which ensures all your bookings are credited appropriately and commissions calculated.

= How do I find my CID? =
Go to the Partner Center (partners.agoda.com) and login/sign up. Then go to Profile > Manage My Sites to see your CIDs listed.

= How can I see how much I will earn in commissions? =
Login to Partner Center and go to Reporting. Any bookings made through your links will appear there within 24 hours. You will earn commission when the user completes his or her stay (at check out).
